Comment faire une lessive?

1 - Trier son linge

Il est recommandé de séparer vos vêtements en piles de tissus épais et de tissus légers ou fins. De cette manière, vous pourrez choisir le cycle de lavage approprié au type de tissu que vous lavez Ensuite, triez-les par couleur pour éviter que les couleurs foncées ne bavent sur les couleurs blanches ou plus claires. Créez une pile de vêtements blancs qui inclue vos t-shirts blancs, vos sous-vêtements blancs et toutes les autres tenues blanches.

2 - Traiter les tâches

Achetez du détergent conçu pour votre type de lave-linge. Certains détergents sont conçus pour les lave-linges traditionnels à chargement par le haut, d’autres sont conçus pour les lave-linges à haute efficacité ou à chargement frontal et d’autres peuvent être utilisés sur les 2 types de machines. Déterminez le type de machine que vous avez et achetez un détergent adapté. Tips : Si vous avez la peau sensible ou si vous êtes sujet(te) aux allergies, achetez du détergent naturel, sans fragrance ou neutre et clair.

3 - Versez de la lessive dans le bac dédié

Pour les machines à laver à chargement par le haut, il est possible que vous deviez d’abord remplir le bac d’eau, verser du détergent puis ajouter le linge. Lisez les instructions dans le couvercle de votre machine pour connaitre la quantité exacte de détergent à ajouter.
